This source discusses how different demographics and states were affected by covid and pneumonia. We used this source to determine how developing pneumonia impacted a COVID-19 paitents survivability.
CDC Provisional COVID-19 data by sex, age, and state reference:
Center for Disease Control. (2020). Provisional COVID-19 Death Counts by Sex, Age, and State. Retrieved 2020, from https://data.cdc.gov/NCHS/Provisional-COVID-19-Death-Counts-by-Sex-Age-and-S/9bhg-hcku

This source shows death counts by sex, age, and state. This source also shows the number of influenza, pneumonia deaths by each of these demographics, and pneumonia deaths that developed from COVID-19. This source will be used to determine how severe developing pneumonia is for patients. 
CDC Provisional Death counts for COVID-19 reference:
Center for Disease Control. (2020, November 11). Provisional Death Counts for Coronavirus Disease (COVID-19). Retrieved December 04, 2020, from https://catalog.data.gov/dataset/provisional-death-counts-for-coronavirus-disease-covid-19

This source compares total number of COVID deaths in a given state and age range with any conditions these patients might have had. Thes include represtory diseases, diabetes, and more. We will use this data to determine the risk of different groups with conditions. Note that people with multiple conditions will be listed in multiple different locations so you cannot summate the total death counts of all conditions, or it will be greater than the total number of COVID deaths.  
Center for Disease Control. (2020). Conditions contributing to deaths involving coronavirus disease 2019 (COVID-19), by age group and state, United States. Retrieved 2020, from https://data.cdc.gov/NCHS/Conditions-contributing-to-deaths-involving-corona/hk9y-quqm
